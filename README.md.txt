Assigment 3

On the Website githup.com/julira you will find a repo called “Getting-and-cleaning Data-Assigment-3”. 
This repo contains the file “run.analysis.R” and the file “CodeBook.md”

The file “run_analysis.R” contains the R script
which can be run in R/R Studio in order to clean the data sets about human activity recognition using Smartphone
collected by Center for Machine Learning and Intellegent System(UCI), and make from the data sets a tidy data set.
 
The file “CodeBook.md” contains the description of the experiment with Smartphone, as well as of data set.
Also the file contains the description of all variables in the merged data set.
